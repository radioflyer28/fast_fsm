# Agent Instructions

All project workflows, issue tracking, quality gates, and session completion rules
live in [`.github/copilot-instructions.md`](.github/copilot-instructions.md).

## Quick Reference

```bash
bd ready                              # Find available work
bd update <id> --status in_progress   # Claim work
bd close <id> --reason "Done"         # Complete work
bd create "Title" -t task -p 2        # Create a task
```

See the copilot instructions for the full development workflow, issue tracking
conventions, and GitHub Issues mirroring policy.
